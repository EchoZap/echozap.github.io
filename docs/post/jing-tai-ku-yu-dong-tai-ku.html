дёҚз®ЎжҳҜLinuxиҝҳжҳҜWindowsдёӯзҡ„еә“ж–Үд»¶е…¶жң¬иҙЁе’Ңе·ҘдҪңжЁЎејҸйғҪжҳҜзӣёеҗҢзҡ„, еҸӘдёҚиҝҮеңЁдёҚеҗҢзҡ„е№іеҸ°дёҠеә“еҜ№еә”зҡ„ж–Үд»¶ж јејҸе’Ңж–Үд»¶еҗҺзјҖдёҚеҗҢгҖӮзЁӢеәҸдёӯи°ғз”Ёзҡ„еә“жңүдёӨз§ҚйқҷжҖҒеә“е’ҢеҠЁжҖҒеә“пјҢдёҚз®ЎжҳҜе“Әз§Қеә“ж–Үд»¶жң¬иҙЁжҳҜиҝҳжҳҜжәҗж–Үд»¶пјҢеҸӘдёҚиҝҮжҳҜдәҢиҝӣеҲ¶ж јејҸеҸӘжңүи®Ўз®—жңәиғҪеӨҹиҜҶеҲ«пјҢдҪңдёәдёҖдёӘжҷ®йҖҡдәәе°ұж— иғҪдёәеҠӣдәҶгҖӮ

еңЁйЎ№зӣ®дёӯдҪҝз”Ёеә“дёҖиҲ¬жңүдёӨдёӘзӣ®зҡ„пјҢдёҖдёӘжҳҜдёәдәҶдҪҝзЁӢеәҸжӣҙеҠ з®ҖжҙҒдёҚйңҖиҰҒеңЁйЎ№зӣ®дёӯз»ҙжҠӨеӨӘеӨҡзҡ„жәҗж–Үд»¶пјҢеҸҰдёҖж–№йқўжҳҜдёәдәҶжәҗд»Јз ҒдҝқеҜҶпјҢжҜ•з«ҹдёҚжҳҜжүҖжңүдәәйғҪжғіжҠҠиҮӘе·ұзј–еҶҷзҡ„зЁӢеәҸејҖжәҗеҮәжқҘгҖӮ

еҪ“жҲ‘д»¬жӢҝеҲ°дәҶеә“ж–Үд»¶пјҲеҠЁжҖҒеә“гҖҒйқҷжҖҒеә“пјүд№ӢеҗҺиҰҒжғідҪҝз”Ёиҝҳеҝ…йЎ»жңүиҝҷдәӣеә“дёӯжҸҗдҫӣзҡ„APIеҮҪж•°зҡ„еЈ°жҳҺпјҢд№ҹе°ұжҳҜеӨҙж–Үд»¶пјҢжҠҠиҝҷдәӣйғҪж·»еҠ еҲ°йЎ№зӣ®дёӯпјҢе°ұеҸҜд»Ҙеҝ«д№җзҡ„еҶҷд»Јз ҒдәҶгҖӮ

1.йқҷжҖҒеә“

еңЁLinuxдёӯйқҷжҖҒеә“з”ұзЁӢеәҸ ar з”ҹжҲҗпјҢзҺ°еңЁйқҷжҖҒеә“е·Із»ҸдёҚеғҸд№ӢеүҚйӮЈд№Ҳжҷ®йҒҚдәҶпјҢиҝҷдё»иҰҒжҳҜз”ұдәҺзЁӢеәҸйғҪеңЁдҪҝз”ЁеҠЁжҖҒеә“гҖӮе…ідәҺйқҷжҖҒеә“зҡ„е‘ҪеҗҚи§„еҲҷеҰӮдёӢ:

  • еңЁLinuxдёӯйқҷжҖҒеә“д»ҘlibдҪңдёәеүҚзјҖ, д»Ҙ.aдҪңдёәеҗҺзјҖ, дёӯй—ҙжҳҜеә“зҡ„еҗҚеӯ—иҮӘе·ұжҢҮе®ҡеҚіеҸҜ, еҚі: libxxx.a
  • еңЁWindowsдёӯйқҷжҖҒеә“дёҖиҲ¬д»ҘlibдҪңдёәеүҚзјҖ, д»ҘlibдҪңдёәеҗҺзјҖ, дёӯй—ҙжҳҜеә“зҡ„еҗҚеӯ—йңҖиҰҒиҮӘе·ұжҢҮе®ҡ, еҚі: libxxx.lib

2.4 и§ЈеҶіеҠЁжҖҒеә“ж— жі•еҠ иҪҪй—®йўҳ

2.4.1 еә“зҡ„е·ҘдҪңеҺҹзҗҶ

  • йқҷжҖҒеә“еҰӮдҪ•иў«еҠ иҪҪ:

    еңЁзЁӢеәҸзј–иҜ‘зҡ„жңҖеҗҺдёҖдёӘйҳ¶ж®өд№ҹе°ұжҳҜй“ҫжҺҘйҳ¶ж®өпјҢжҸҗдҫӣзҡ„йқҷжҖҒеә“дјҡиў«жү“еҢ…еҲ°еҸҜжү§иЎҢзЁӢеәҸдёӯгҖӮеҪ“еҸҜжү§иЎҢзЁӢеәҸиў«жү§иЎҢпјҢйқҷжҖҒеә“дёӯзҡ„д»Јз Ғд№ҹдјҡдёҖе№¶иў«еҠ иҪҪеҲ°еҶ…еӯҳдёӯпјҢеӣ жӯӨдёҚдјҡеҮәзҺ°йқҷжҖҒеә“жүҫдёҚеҲ°ж— жі•иў«еҠ иҪҪзҡ„й—®йўҳгҖӮ

  • еҠЁжҖҒеә“еҰӮдҪ•иў«еҠ иҪҪ

    • еңЁзЁӢеәҸзј–иҜ‘зҡ„жңҖеҗҺдёҖдёӘйҳ¶ж®өд№ҹе°ұжҳҜй“ҫжҺҘйҳ¶ж®өпјҡ

      • еңЁgccе‘Ҫд»ӨдёӯиҷҪз„¶жҢҮе®ҡдәҶеә“и·Ҝеҫ„(дҪҝз”ЁеҸӮж•° -L ), дҪҶжҳҜиҝҷдёӘи·Ҝеҫ„е№¶жІЎжңүи®°еҪ•еҲ°еҸҜжү§иЎҢзЁӢеәҸдёӯпјҢеҸӘжҳҜжЈҖжҹҘдәҶиҝҷдёӘи·Ҝеҫ„дёӢзҡ„еә“ж–Үд»¶жҳҜеҗҰеӯҳеңЁгҖӮ
      • еҗҢж ·еҜ№еә”зҡ„еҠЁжҖҒеә“ж–Үд»¶д№ҹжІЎжңүиў«жү“еҢ…еҲ°еҸҜжү§иЎҢзЁӢеәҸдёӯпјҢеҸӘжҳҜеңЁеҸҜжү§иЎҢзЁӢеәҸдёӯи®°еҪ•дәҶеә“зҡ„еҗҚеӯ—гҖӮ

    • еҸҜжү§иЎҢзЁӢеәҸиў«жү§иЎҢиө·жқҘд№ӢеҗҺ:

      • зЁӢеәҸжү§иЎҢзҡ„ж—¶еҖҷдјҡе…ҲжЈҖжөӢйңҖиҰҒзҡ„еҠЁжҖҒеә“жҳҜеҗҰеҸҜд»Ҙиў«еҠ иҪҪпјҢеҠ иҪҪдёҚеҲ°е°ұдјҡжҸҗзӨәдёҠиҫ№зҡ„й”ҷиҜҜдҝЎжҒҜ
      • еҪ“еҠЁжҖҒеә“дёӯзҡ„еҮҪж•°еңЁзЁӢеәҸдёӯиў«и°ғз”ЁдәҶ, иҝҷдёӘж—¶еҖҷеҠЁжҖҒеә“жүҚеҠ иҪҪеҲ°еҶ…еӯҳпјҢеҰӮжһңдёҚиў«и°ғз”Ёе°ұдёҚеҠ иҪҪ
      • еҠЁжҖҒеә“зҡ„жЈҖжөӢе’ҢеҶ…еӯҳеҠ иҪҪж“ҚдҪңйғҪжҳҜз”ұеҠЁжҖҒиҝһжҺҘеҷЁжқҘе®ҢжҲҗзҡ„

2.4.2 еҠЁжҖҒй“ҫжҺҘеҷЁ

еҠЁжҖҒй“ҫжҺҘеҷЁжҳҜдёҖдёӘзӢ¬з«ӢдәҺеә”з”ЁзЁӢеәҸзҡ„иҝӣзЁӢ, еұһдәҺж“ҚдҪңзі»з»ҹ, еҪ“з”ЁжҲ·зҡ„зЁӢеәҸйңҖиҰҒеҠ иҪҪеҠЁжҖҒеә“зҡ„ж—¶еҖҷеҠЁжҖҒиҝһжҺҘеҷЁе°ұејҖе§Ӣе·ҘдҪңдәҶпјҢеҫҲжҳҫз„¶еҠЁжҖҒиҝһжҺҘеҷЁж №жң¬е°ұдёҚзҹҘйҒ“з”ЁжҲ·йҖҡиҝҮ gcc зј–иҜ‘зЁӢеәҸзҡ„ж—¶еҖҷйҖҡиҝҮеҸӮж•° -LжҢҮе®ҡзҡ„и·Ҝеҫ„гҖӮ

йӮЈд№ҲеҠЁжҖҒй“ҫжҺҘеҷЁжҳҜеҰӮдҪ•жҗңзҙўжҹҗдёҖдёӘеҠЁжҖҒеә“зҡ„е‘ўпјҢеңЁе®ғеҶ…йғЁжңүдёҖдёӘй»ҳи®Өзҡ„жҗңзҙўйЎәеәҸпјҢжҢүз…§дјҳе…Ҳзә§д»Һй«ҳеҲ°дҪҺзҡ„йЎәеәҸеҲҶеҲ«жҳҜпјҡ

1.еҸҜжү§иЎҢж–Үд»¶еҶ…йғЁзҡ„ DT_RPATH ж®ө

2.зі»з»ҹзҡ„зҺҜеўғеҸҳйҮҸ LD_LIBRARY_PATH

3.зі»з»ҹеҠЁжҖҒеә“зҡ„зј“еӯҳж–Үд»¶ /etc/ld.so.cache

4.еӯҳеӮЁеҠЁжҖҒеә“/йқҷжҖҒеә“зҡ„зі»з»ҹзӣ®еҪ• /lib/, /usr/libзӯү
жҢүз…§д»ҘдёҠеӣӣдёӘйЎәеәҸ, дҫқж¬Ўжҗңзҙў, жүҫеҲ°д№ӢеҗҺз»“жқҹйҒҚеҺҶ, жңҖз»ҲиҝҳжҳҜжІЎжүҫеҲ°, еҠЁжҖҒиҝһжҺҘеҷЁе°ұдјҡжҸҗзӨәеҠЁжҖҒеә“жүҫдёҚеҲ°зҡ„й”ҷиҜҜдҝЎжҒҜгҖӮ

2.4.3 и§ЈеҶіж–№жЎҲ

еҸҜжү§иЎҢзЁӢеәҸз”ҹжҲҗд№ӢеҗҺ, ж №жҚ®еҠЁжҖҒй“ҫжҺҘеҷЁзҡ„жҗңзҙўи·Ҝеҫ„, жҲ‘д»¬еҸҜд»ҘжҸҗдҫӣдёүз§Қи§ЈеҶіж–№жЎҲпјҢжҲ‘д»¬еҸӘйңҖиҰҒе°ҶеҠЁжҖҒеә“зҡ„и·Ҝеҫ„ж”ҫеҲ°еҜ№еә”зҡ„зҺҜеўғеҸҳйҮҸжҲ–иҖ…зі»з»ҹй…ҚзҪ®ж–Үд»¶дёӯпјҢеҗҢж ·д№ҹеҸҜд»Ҙе°ҶеҠЁжҖҒеә“жӢ·иҙқеҲ°зі»з»ҹеә“зӣ®еҪ•пјҲжҲ–иҖ…жҳҜе°ҶеҠЁжҖҒеә“зҡ„иҪҜй“ҫжҺҘж–Үд»¶ж”ҫеҲ°иҝҷдәӣзі»з»ҹеә“зӣ®еҪ•дёӯпјүгҖӮ

ж–№жЎҲ1

е°Ҷеә“и·Ҝеҫ„ж·»еҠ еҲ°зҺҜеўғеҸҳйҮҸ LD_LIBRARY_PATH дёӯ

1.жүҫеҲ°зӣёе…ізҡ„й…ҚзҪ®ж–Үд»¶
з”ЁжҲ·зә§еҲ«: ~/.bashrc вҖ”> и®ҫзҪ®еҜ№еҪ“еүҚз”ЁжҲ·жңүж•Ҳ
зі»з»ҹзә§еҲ«: /etc/profile вҖ”> и®ҫзҪ®еҜ№жүҖжңүз”ЁжҲ·жңүж•Ҳ

2.дҪҝз”Ё vim жү“ејҖй…ҚзҪ®ж–Үд»¶, еңЁж–Үд»¶жңҖеҗҺж·»еҠ иҝҷж ·дёҖеҸҘиҜқ

# иҮӘе·ұжҠҠи·Ҝеҫ„еҶҷиҝӣеҺ»е°ұиЎҢдәҶ
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:еҠЁжҖҒеә“зҡ„з»қеҜ№и·Ҝеҫ„

3.и®©дҝ®ж”№зҡ„й…ҚзҪ®ж–Үд»¶з”ҹж•Ҳ

  • дҝ®ж”№дәҶз”ЁжҲ·зә§еҲ«зҡ„й…ҚзҪ®ж–Үд»¶, е…ій—ӯеҪ“еүҚз»Ҳз«Ҝ, жү“ејҖдёҖдёӘж–°зҡ„з»Ҳз«Ҝй…ҚзҪ®е°ұз”ҹж•ҲдәҶ
  • дҝ®ж”№дәҶзі»з»ҹзә§еҲ«зҡ„й…ҚзҪ®ж–Үд»¶, жіЁй”ҖжҲ–е…ій—ӯзі»з»ҹ, еҶҚејҖжңәй…ҚзҪ®е°ұз”ҹж•ҲдәҶ
  • дёҚжғіжү§иЎҢдёҠиҫ№зҡ„ж“ҚдҪң, еҸҜд»Ҙжү§иЎҢдёҖдёӘе‘Ҫд»Өи®©й…ҚзҪ®йҮҚж–°иў«еҠ иҪҪ
# дҝ®ж”№зҡ„жҳҜе“ӘдёҖдёӘе°ұжү§иЎҢеҜ№еә”зҡ„йӮЈдёӘе‘Ҫд»Ө
# source еҸҜд»Ҙз®ҖеҶҷдёәдёҖдёӘ . , дҪңз”ЁжҳҜи®©ж–Үд»¶еҶ…е®№иў«йҮҚж–°еҠ иҪҪ
$ source ~/.bashrc          (. ~/.bashrc)
$ source /etc/profile       (. /etc/profile)
ж–№жЎҲ2

жӣҙж–° /etc/ld.so.cache ж–Үд»¶

1.жүҫеҲ°еҠЁжҖҒеә“жүҖеңЁзҡ„з»қеҜ№и·Ҝеҫ„пјҲдёҚеҢ…жӢ¬еә“зҡ„еҗҚеӯ—пјүжҜ”еҰӮпјҡ/home/robin/Library/

2.дҪҝз”Ёvim дҝ®ж”№ /etc/ld.so.conf иҝҷдёӘж–Үд»¶, е°ҶдёҠиҫ№зҡ„и·Ҝеҫ„ж·»еҠ еҲ°ж–Үд»¶дёӯ(зӢ¬иҮӘеҚ дёҖиЎҢ)

# 1. жү“ејҖж–Үд»¶
sudo vim /etc/ld.so.conf

# 2. ж·»еҠ еҠЁжҖҒеә“и·Ҝеҫ„, е№¶дҝқеӯҳйҖҖеҮә

3.жӣҙж–° /etc/ld.so.confдёӯзҡ„ж•°жҚ®еҲ° /etc/ld.so.cache дёӯ

# еҝ…йЎ»дҪҝз”Ёз®ЎзҗҶе‘ҳжқғйҷҗжү§иЎҢиҝҷдёӘе‘Ҫд»Ө
sudo ldconfig
ж–№жЎҲ3

жӢ·иҙқеҠЁжҖҒеә“ж–Үд»¶еҲ°зі»з»ҹеә“зӣ®еҪ• /lib/ жҲ–иҖ… /usr/lib дёӯ (жҲ–иҖ…е°Ҷеә“зҡ„иҪҜй“ҫжҺҘж–Үд»¶ж”ҫиҝӣеҺ»)пјҢд»ҘдёӢдәҢйҖүдёҖ

# еә“жӢ·иҙқ
sudo cp /xxx/xxx/libxxx.so /usr/lib

# еҲӣе»әиҪҜиҝһжҺҘ
sudo ln -s /xxx/xxx/libxxx.so /usr/lib/libxxx.so
ж–№жЎҲ4

еҰӮжһңдёҠиҝ°ж–№жі•д»Қз„¶дёҚиЎҢпјҢеҸҜд»ҘеңЁиҝҗиЎҢж—¶жҢҮе®ҡжҗңзҙўи·Ҝеҫ„пјҡе°қиҜ•еңЁзј–иҜ‘ж—¶дҪҝз”Ё -Wl,-rpath йҖүйЎ№жҢҮе®ҡе…ұдә«еә“зҡ„иҝҗиЎҢж—¶жҗңзҙўи·Ҝеҫ„пјҢдҫӢеҰӮпјҡ

gcc main.c -o app -L./ -ltest -Wl,-rpath,/path/library

2.4.4 йӘҢиҜҒ

еңЁеҗҜеҠЁеҸҜжү§иЎҢзЁӢеәҸд№ӢеүҚ, жҲ–иҖ…еңЁи®ҫзҪ®дәҶеҠЁжҖҒеә“и·Ҝеҫ„д№ӢеҗҺ, жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮдёҖдёӘе‘Ҫд»ӨжЈҖжөӢзЁӢеәҸиғҪдёҚиғҪеӨҹйҖҡиҝҮеҠЁжҖҒй“ҫжҺҘеҷЁеҠ иҪҪеҲ°еҜ№еә”зҡ„еҠЁжҖҒеә“, иҝҷдёӘе‘Ҫд»ӨеҸ«еҒҡ ldd

3.дјҳзјәзӮ№

3.1 йқҷжҖҒеә“

дјҳзӮ№пјҡ

  • йқҷжҖҒеә“иў«жү“еҢ…еҲ°еә”з”ЁзЁӢеәҸдёӯеҠ иҪҪйҖҹеәҰеҝ«
  • еҸ‘еёғзЁӢеәҸж— йңҖжҸҗдҫӣйқҷжҖҒеә“пјҢз§»жӨҚж–№дҫҝ

зјәзӮ№пјҡ

  • зӣёеҗҢзҡ„еә“ж–Үд»¶ж•°жҚ®еҸҜиғҪеңЁеҶ…еӯҳдёӯиў«еҠ иҪҪеӨҡд»Ҫ, ж¶ҲиҖ—зі»з»ҹиө„жәҗпјҢжөӘиҙ№еҶ…еӯҳ
  • еә“ж–Үд»¶жӣҙж–°йңҖиҰҒйҮҚж–°зј–иҜ‘йЎ№зӣ®ж–Үд»¶, з”ҹжҲҗж–°зҡ„еҸҜжү§иЎҢзЁӢеәҸ, жөӘиҙ№ж—¶й—ҙгҖӮ

3.2 еҠЁжҖҒеә“

дјҳзӮ№пјҡ

  • еҸҜе®һзҺ°дёҚеҗҢиҝӣзЁӢй—ҙзҡ„иө„жәҗе…ұдә«
  • еҠЁжҖҒеә“еҚҮзә§з®ҖеҚ•, еҸӘйңҖиҰҒжӣҝжҚўеә“ж–Үд»¶, ж— йңҖйҮҚж–°зј–иҜ‘еә”з”ЁзЁӢеәҸ
  • зЁӢеәҸзҢҝеҸҜд»ҘжҺ§еҲ¶дҪ•ж—¶еҠ иҪҪеҠЁжҖҒеә“, дёҚи°ғз”Ёеә“еҮҪж•°еҠЁжҖҒеә“дёҚдјҡиў«еҠ иҪҪ

зјәзӮ№пјҡ

  • еҠ иҪҪйҖҹеәҰжҜ”йқҷжҖҒеә“ж…ў, д»ҘзҺ°еңЁи®Ўз®—жңәзҡ„жҖ§иғҪеҸҜд»ҘеҝҪз•Ҙ
  • еҸ‘еёғзЁӢеәҸйңҖиҰҒжҸҗдҫӣдҫқиө–зҡ„еҠЁжҖҒеә“
